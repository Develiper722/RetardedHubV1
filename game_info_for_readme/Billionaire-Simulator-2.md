# Features 

## Auto Work
- 9 separate toggles for auto work on each station (`SW 1` → `SW 9`)
- Each toggle runs independently; multiple stations can auto-work at once

## Auto Upgrade
- 9 toggles per station for each upgrade type:
  - `SU1` → single-level upgrade
  - `SU10` → 10-level upgrade
  - `SU100` → 100-level upgrade
- Independent per station and per type

## Control Tab
- **Stop All** button — stops all auto work and upgrades instantly
- **Debug Logs toggle** — prints errors or remote failures to console if enabled
